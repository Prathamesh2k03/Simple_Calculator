# 🧮 Calculator Web App

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations with a clean user interface.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧹 Clear display
* ⌫ Delete last digit
* ⚠️ Error handling

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

```
calculator/
│── index.html
│── style.css
│── script.js
```

---

## ▶️ How to Run

1. Download or clone this repository
2. Open the project folder
3. Double-click `index.html`

OR use **Live Server in VS Code**
